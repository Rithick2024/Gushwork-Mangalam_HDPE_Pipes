## 📌 Features

### 🧷 Sticky Header
- Appears after scrolling past the first fold
- Smooth transition and animation
- Positioned above the navigation bar
- Hides when scrolling back up

### 🖼️ Image Carousel with Zoom
- Interactive horizontal carousel
- Hover-based zoom preview for images
- Smooth transitions and hover effects
- Matches Figma interaction behavior

### 📱 Fully Responsive Design
- Desktop, tablet, and mobile optimized
- Layout adapts using Flexbox and Grid
- Consistent spacing and typography across devices
